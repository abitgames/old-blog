ex2D 1.1.3, TileMap Editor And Gui Border Component Preview!

Written on September 20, 2011 by nantas in Blog

Another ex2D version released! Our programmer Karas is on fire xD

As usual follow this link for more information and more links!

Changes in v 1.1.3Â 

New Features

  • Add exCollisionHelper Component

Changes

  • Add useBuildColor property in exAtlasInfo
  • Add Anchor(StartFrom) property in exScreenPosition (github issue)
  • Auto Resize Box-Collider will fixed the thick of the box to camera.farClip – camera.nearClip
  • Add Preview Size in sprite animation editor

Bugs Fixes

    • Fix buildColor not keep color in transparent area
    • Fix SpriteFont multiline alignment not correct when scale larger than 1.0
    • Fix SpriteAnimation may not trigger event at last frame
    • Fix SpriteAnimation trigger Play event (switch to other sprite animation), the first frame event in target animation will not play
    • Fix Mesh will not release when sprite destroyed
    • Fix sprite animation add clip will not update textures in empty sprite
    • Fix exBitmapFont can not created from font page with white space in its name
    • Fix exSprite, exSpriteFont rebuild will change the Non-AutoResize collision box
  • Fix sprite animation not update vertex when the two frame have the same width and height but use texture offset
But hold on my dear friends! This is not even the exciting part of this blog post!
What we want to share today is two screenshots of upcoming ex2D feature and add-on in development!

GUI Border Component

We lied about not going to make a component for GUI stuff. But this Scale9grid implementation is really needed for our current project so here it is! By looking at the screenshot you should get the idea how convenient this tool can be and how much texture space you could save by using this component to create GUI panels.

This feature will be available in the next release version of ex2D. I can’t wait to create some sleek GUI with it!

TileMap Editor (add-on)

Here we go TileMap editor! Create your 2D platform or RPG game with ex2D with ease! We will release more details about feature and usage in the future. But anyway it’s coming and please be prepared.

Also worth mentioning is that TileMap Editor will not be included in the ex2D core package. We will sell this in its own add-on package that requires ex2D core package to run. We are doing this to make sure nobody pays for features they don’t need. And the add-on price will be quite affordable as well.

That’s for today boys and girls. Please subscribe our RSS or twitter to make sure you receive the updates before everyone else! <3
